Installation

There is no need of any installation like npm i <package-name> or also no need any CDN.

As we know it is made in only HTML, CSS and JavaScript. So you have to just copy and paste the codes into your projects.


This site is using "Ionicons" library for icons so if also looking for same icon then pleae make sure you are imported following CDN.


CDNs:

<script type="module" src="https://unpkg.com/ionicons@7.1.0/dist/ionicons/ionicons.esm.js" ></script>
<script nomodule src="https://unpkg.com/ionicons@7.1.0/dist/ionicons/ionicons.js" ></script>
